# Calculatoe

Calculatoe is a web-based Tic-Tac-Toe game that adds a fun and educational twist by incorporating calculus questions into the gameplay. Players must solve calculus problems before placing their marks on the board, making this game both challenging and a great learning tool for math enthusiasts.

## Features
- **Player vs Player Mode**: Two players face off, and each must solve a calculus equation before making a move.
- **Player vs Bot Mode**: Play against a bot that uses an algorithm according to the chosen difficulty for optimal gameplay.
- **Random Calculus Questions**: Each cell of the Tic-Tac-Toe board contains a calculus question that must be answered correctly before a move can be made.
- **Simple, Intuitive Interface**: Built with HTML, CSS, and JavaScript for a clean and easy-to-use design.

## How to Play
1. Choose between **Player vs Player** or **Player vs Bot** mode.
2. On each turn, solve the calculus problem presented in the cell you want to mark.
3. If you answer correctly, your mark (X or O) will be placed in the cell.
4. The first player to get three marks in a row (horizontally, vertically, or diagonally) wins!
